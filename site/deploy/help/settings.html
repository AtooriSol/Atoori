The "Settings" dialog allows for the viewing and modification of various aspects of the JS7800 emulator. Any changes that are made in the dialog can be persisted by clicking the "OK" button. Clicking the "Cancel" button will discard any pending changes that have been made. The "Defaults" button can be used to revert all settings back to their original default values.

Settings values will persist between sessions unless cookies have been explicitly disabled.

Display Settings

The "Display" settings tab (shown below) is used to control how the JS7800 screen will be displayed.

Each of the fields present on the "Display" settings tab are described below.

Screen size
The screen size setting controls the size of the JS7800 screen. While there are many size settings available, use of an integer-based value (2x, 3x, etc.) is recommended as it supports a crisp display (apply filtering disabled) without fractional pixels.
Aspect ratio
The aspect ratio (AR) setting controls the ratio of the screen width to its height. By default, a 1:1 pixel aspect ratio (PAR) is selected that supports a crisp display (apply filtering disabled) without fractional pixels. Other aspect ratios are available, including a 6:7 PAR option that attempts to mimic the original ratio of the Atari 7800 console. However, ratios other than a 1:1 PAR will result in fractional pixels, therefore filtering of the display (apply filter) is recommended.
Apply filter
This setting controls whether a filter is applied to the display. The actual filter applied depends on the particular browser and version (typically a bilinear filter). Applying a filter will cause the screen to have a slightly blurred appearance, but eliminates the majority of artifacts that result from fractional pixels.
